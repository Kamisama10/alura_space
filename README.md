Este é um projeto desenvolvido durante o curso de Django: modelos, rotas e views da Alura. A aplicação consiste em uma galeria de fotografias do espaço onde é possível visualizar imagens. Tecnologias Utilizadas: Python, Django 4.x/5.x (Web Framework) HTML/CSS (Templates e Estilização) SQLite (Banco de dados padrão do Django).

Funcionalidades
Organização por Templates: Uso de extends e include para reaproveitamento de código (DRY - Don't Repeat Yourself).
Gestão de Arquivos Estáticos: Configuração correta de CSS e imagens utilizando a tag {% static %}.
Rotas Dinâmicas: Sistema de URLs amigáveis e organizadas por apps.
Cards Interativos: Exibição de informações vindas do banco de dados (ou dicionários de dados).

Boas Práticas Aplicadas
Arquitetura MVT: Separação clara entre Models, Views e Templates.
Static Files: Uso rigoroso da tag {% static %} para garantir a portabilidade do projeto.
Nomenclatura: Variáveis e funções nomeadas em português/inglês seguindo a consistência do curso.
